    Buttons

    There are six different default button styles. Each button style is meant for a different situation such as the banners. 

                        
//There are two standard buttons,
//"button-standard" & "button-standard-dark"
button class="button-standard"
    hello World
/button
                    
                
                            
//other button types are used in
//a similar manner, but there are
//more variations.
//they are: "button-succes",
//"button-Prime", "button-Danger"
//"button-Warning", & "button-Info"

button class="button-succes"
    hello World
/button
