    • Colors and text

    The background and text color of every element is customizeable, as well as heights and text alignment. 

                        
h3 class="cntr bkgr_W border txt_G Height_200"
    This is the example above with 
    all the tags set up
/h3
// "bkgr_W" stands for background white.
// the different background types are
// "bkgr_W", "bkgr_B", "bkgr_Blue", "bkgr_G",
// "bkgr_R"
// text color has a similar set up with the
// change of "bkgr" to "txt". example "txt_W"

//Text can also be centered by adding the 
//"cntr" class tag

//Borders can be added by adding the 
//"border" class tag

//Lastly, you can change the height of
//an element by adding the "Height" 
//tag class and adding a height at an
//inncrements of 40 from 40 to 400.
